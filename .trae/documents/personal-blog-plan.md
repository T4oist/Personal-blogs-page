# 个人博客网站开发计划

## 项目概述
为 **T4oist** 创建一个具有强烈个人风格的现代博客网站，参考 antfu.me、innei.in 等优秀个人博客的设计风格。

## 动效设计方案

### 页面过渡动画
- **路由切换**：淡入淡出 + 轻微滑动
- **内容加载**：骨架屏 + 渐显效果

### 微交互效果
| 元素 | 动效 |
|------|------|
| 卡片悬停 | 上浮 + 阴影加深 + 边框发光 |
| 按钮悬停 | 背景色渐变 + 轻微缩放 |
| 链接悬停 | 下划线展开动画 |
| 图片悬停 | 轻微放大 + 滤镜效果 |
| 标签悬停 | 背景色填充动画 |

### 滚动动画
- **进入视口**：元素从下方淡入
- **视差效果**：背景元素轻微移动
- **进度指示**：阅读进度条

### 主题切换
- 平滑的颜色过渡动画
- 图标旋转切换效果

## 博客内容管理说明

### 新建文章
在 `src/content/blog/` 目录下创建 `.md` 文件：

```markdown
---
title: 文章标题
description: 文章描述
date: 2025-03-31
tags: [标签1, 标签2]
category: 分类名
---

文章内容...
```

### 新建碎念
在 `src/content/moments/` 目录下创建 `.md` 文件：

```markdown
---
date: 2025-03-31
---

碎念内容...
```

### 修改内容
直接编辑对应的 `.md` 文件，Git 提交后自动部署更新。
